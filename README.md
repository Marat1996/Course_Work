# Название проекта

Курсовой проект.

## Описание
Программа позволяет собирать информацию с двух сайтов: HeadHunter и SuperJob.
Сбор информации ведется по ключевому слову, пользователю нужно выбрать платформу и указать минимальную заработную плату.
